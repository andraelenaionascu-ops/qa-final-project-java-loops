# 🚀 Java Loops & User Validation Project
![Java CI with Maven](https://github.com/andraelenaionascu-ops/qa-final-project-java-loops/actions/workflows/ci.yml/badge.svg)

Acest proiect reprezintă o aplicație de tip consolă realizată în Java, care simulează procesul de creare a unui cont de utilizator prin validarea unui **Username**.

## 📋 Cerințe de Validare
Programul verifică dacă username-ul introdus respectă următoarele reguli:
1. **Lungime:** Trebuie să aibă între **6 și 12 caractere** (inclusiv).
2. **Spații:** **Nu trebuie** să conțină spații.
3. **Cifre:** Trebuie să conțină **cel puțin o cifră (0-9)**.

## 🛠️ Tehnologii Utilizate
* **Java 17** (LTS)
* **Maven** (Managementul dependențelor și build)
* **GitHub Actions** (Pipeline pentru testare automată la fiecare push)
* **Git** (Version Control)
